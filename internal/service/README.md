# Бизнес-логика

Сервисы:
- **TeamService** - управление командами
- **PullRequestService** - создание PR, мерж, перераспределение ревьюверов
- **UserService** - управление статусом пользователей
- **StatsService** - статистика по назначениям

вспомогательное:
- **dependencies** - структура со всеми сервисами
