# Внутренняя логика приложения

## Основные модули:
- **app/** - инициализация и запуск приложения
- **config/** - конфигурация через env-переменные
- **domain/** - доменные модели и ошибки
- **repository/** - слой работы с БД
- **service/** - бизнес-логика
- **transport/** - HTTP-обработчики и роутинг

## Поток данных:
HTTP → transport → service → repository → DB
